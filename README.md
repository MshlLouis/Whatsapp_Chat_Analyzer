# A program to summarize a Whatsapp Chat into a JSON file, display them on a Website and download as an image


## 1: Requirements

- IDE with JDK 21+ & Gradle 8+
- WinRAR or 7zip


## 2: Getting Started

1. First you will need to export your chat on your phone and copy the .zip on your PC. It is recommended to export with media to support upcoming versions, but as of the initial release it doesn't matter.
Next you want to unzip your files into a folder. Make sure to do this in a folder that you can easily access. Once done, you can proceed to the next step.
2. Import the project files into an IDE of your liking. The program was written using JDK 24 and Gradle 8.14.5, so if you decide to use different versions, make sure to update your files to use correct versions.
3. You'll find the main file in "src/main/java" called "mainfile.java". This is the only file you'll have to modify. Scroll all the way to the top and you'll see the variables that you can change along with comments on what to put exactly.
4. Once everything is filled out, execute the program. You'll get a JSON file containing all relevant information.
5. Now open the folder "Website" and then "index.html". It opens a local website with standard values for all parameters inside the JSON file. Simply click the "Choose file" button on the top right and select your own file.
6. Finally you can download everything as an image using the "Save as Image" button on the top left.
